Key Technical Decisions
- Single HTML file -- self-contained, zero dependencies, open in any browser
- Google Fonts loaded via CDN (Cinzel Decorative, Quicksand, JetBrains Mono)
- CSS custom properties for the entire color system (easy theming)
- CSS animations for all effects (no JS animation libraries)
- localStorage for save/load
- No audio files -- all feedback is visual (keeps file self-contained)
- Responsive -- works on desktop, tablet, and phone
- Math rendering -- HTML entities + CSS for fractions/exponents (no MathJax dependency)
